# SPLTJFI

## What is SPLTJFI?
SPLTJFI is an App for managing Employee Overtime data, where Overtime Orders are made digitally and ACC by the superior also digitally without having to sign.

## Framework Used?
Codeigniter 4

## Database Used?
MySQL
